## ğŸ“š Idea/Concept

The C4 Model is a hierarchical approach to software architecture documentation created by Simon Brown. It provides four levels of abstraction (Context, Container, Component, Code) that allow different audiences to understand the system at appropriate levels of detail. The model uses simple, clear diagrams that can be understood by both technical and non-technical stakeholders.
