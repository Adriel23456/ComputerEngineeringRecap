The history of compilers traces the evolution of software that transforms human-readable code into machine-executable instructions, marking pivotal advances in computing that enabled modern programming.

### The Beginning: ENIAC and Manual Programming (1946)

ENIAC (Electronic Numerical Integrator and Computer) was programmed by physically setting switches and rewiring connections. Programming meant:
- Setting thousands of switches by hand
- Connecting cables to route data
- No software layer existed between human intent and hardware

### Grace Hopper and the A-0 System (1952)

Grace Hopper developed the A-0 System, the first compiler-like tool:
- Translated mathematical notation into machine code
- Introduced the concept of automatic code generation
- Called it a "compiler" because it compiled subroutines from a library

### FORTRAN: The First True Compiler (1954-1957)

John Backus led the IBM team that created FORTRAN (FORmula TRANslation):
- First high-level programming language with a compiler
- Took 18 person-years to develop
- Generated code nearly as efficient as hand-written assembly
- Proved that compilers could be practical for real-world use

### Evolution Timeline

| Year | Development | Significance |
|------|-------------|--------------|
| 1946 | ENIAC | First electronic general-purpose computer |
| 1952 | A-0 System | First automatic code translator |
| 1957 | FORTRAN | First optimizing compiler |
| 1958 | LISP | First compiler for functional language |
| 1960 | COBOL | First business-oriented compiled language |
| 1970s | C Compiler | Portable compiler design (Unix) |
| 1987 | GCC | GNU Compiler Collection released |
| 2003 | LLVM | Modular compiler infrastructure |

### Key Contributions

Grace Hopper's Philosophy:
> "Programming languages should be closer to English, making them accessible to more people."

John Backus's Vision:
- Automatic programming was not just possible but practical
- Compilers could optimize better than many human programmers
- High-level languages would dominate software development
